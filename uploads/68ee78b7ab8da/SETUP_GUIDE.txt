================================================================================
                    FRIENDSHARE - QUICK SETUP GUIDE
================================================================================

📋 STEP-BY-STEP INSTALLATION:

1. DATABASE SETUP
   ─────────────────────────────────────────────────────────────────────────
   a) Start XAMPP Control Panel
   b) Start Apache and MySQL services
   c) Open phpMyAdmin: http://localhost/phpmyadmin
   d) Click "Import" tab
   e) Click "Choose File" and select: database.sql
   f) Click "Go" button
   ✓ Database "friendshare" will be created with users and files tables

2. VERIFY CONFIGURATION
   ─────────────────────────────────────────────────────────────────────────
   a) Open: includes/db.php
   b) Check database credentials:
      - Host: localhost
      - Database: friendshare
      - Username: root
      - Password: (empty by default)
   c) Update if your MySQL credentials are different

3. FILE PERMISSIONS
   ─────────────────────────────────────────────────────────────────────────
   The uploads/ folder must be writable:
   - On Windows: Already configured (no action needed)
   - On Linux/Mac: Run: chmod 755 uploads/

4. ACCESS THE APPLICATION
   ─────────────────────────────────────────────────────────────────────────
   Open your browser and go to:
   
   🌐 http://localhost/file/
   
   You'll be redirected to the login page.

5. CREATE YOUR FIRST ACCOUNT
   ─────────────────────────────────────────────────────────────────────────
   a) Click "Register here"
   b) Fill in:
      - Full Name: Your Name
      - Email: your@email.com
      - Password: minimum 6 characters
      - Confirm Password: same as above
   c) Click "Register"
   d) You'll be redirected to login page

6. LOGIN AND START SHARING
   ─────────────────────────────────────────────────────────────────────────
   a) Enter your email and password
   b) Click "Login"
   c) You're now on the Dashboard!

7. TEST FILE SHARING
   ─────────────────────────────────────────────────────────────────────────
   To test file sharing, you need at least 2 users:
   
   a) Create a second account:
      - Logout (top-right corner)
      - Register another user
   
   b) Login with first account
   
   c) Share a file:
      - Select the second user from dropdown
      - Choose a file (max 10MB)
      - Click "Upload & Share"
   
   d) Login with second account
   
   e) Download the file:
      - See file in "Received Files" section
      - Click "Download" button

================================================================================
                            TROUBLESHOOTING
================================================================================

❌ "Database Connection Failed"
   → Ensure MySQL is running in XAMPP
   → Check credentials in includes/db.php
   → Verify database "friendshare" exists

❌ "Failed to upload file"
   → Check uploads/ folder exists
   → Verify folder has write permissions
   → Ensure file size is under 10MB

❌ Blank page or errors
   → Enable error reporting in php.ini:
     display_errors = On
     error_reporting = E_ALL
   → Restart Apache after changes

❌ Session issues
   → Clear browser cookies
   → Check session.save_path in php.ini
   → Restart Apache
